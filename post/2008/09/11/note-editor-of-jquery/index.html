（筆記）撰寫jQuery的工具

發表於 2008-09-11 |

說實在我還是比較習慣用文字編輯器就好，目前最愛用的是emeditor和editor plus，不過哪天工作會大量用到dreamweaver或eclipse時就有可能會用到了。

Dreamwaver擴充工具(原文連結)

下載 jQuery_API.mxp
然後在 DW –>命令–>管理擴充功能–>CTRL+I 選擇 jQuery_API.mxp –>勾選開啟

如果沒有管理擴充功能-請至

http://www.adobe.com/tw/exchange/em_download/ 下載安裝即可以



APTANA擴充工具(原文連結)
推薦jQuery的編寫(含HTML 及CSS)使用aptana Studio

Step 1:

至 http://www.aptana.com/studio/download 下載 aptana (約91mb)

Step2:

安裝 aptana Studio

Step 3:

啟動 aptana Studio –>window–>preferences
—>Help—->Install/Update

將 Update Policy 的Policy URL 加入 http://update.aptana.com/install/studio/3.2/

Step 4: aptana Studio –>window–>preferences–>General–>Content
Types
按 text 在下方加入 (ADD) .html 及 .js
Default encoding 輸入 utf-8 (為了中文輸入)

完成後等aptana更新下載一些相關的Ajax Framework (包括jQuery 1.2)重開 aptana

aptana Studio –>script–>References

檢查 jQuery 是否在其中

Step 5:

aptana Studio –>window–>preferences
—->Aptana–>Editors—

>Javascript—>Code Assist

將 jQuery 1.2 勾選起來

打開一個檔案，輸入jQuery的程式碼就會在輸入中出現jQuery的tooltip (Code Assist)

%%如果設定後仍不可輸入中文，使用其他的中文輸入法(如 Yahoo 奇摩輸入法)

http://tw.download.yahoo.com/keykey/
