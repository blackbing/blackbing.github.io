情人節APP

發表於 2011-08-08 |

　 

今年情人節突發奇想的搞了個APP遊戲給我家女王玩，順便熟悉一下jQuery Mobile。先講一下這個遊戲的流程。

地點


這次的地點我選在離家近的中和環球購物中心，因為對我來說場勘也比較容易。哈哈。進去之後，營造出你已經踏入遊戲世界，請拿出你的通話裝置，遊戲即將開始。

掃描QR Code


下載遊戲程式，請事先檢查有沒有安裝掃QR code軟體。以及環境網路暢通。

安裝APK


記得要是先將女王的Android手機的設定改成「允許接受未知的來源」，不然安裝時還要改來改去就糗大了。

遊戲開始


大方的讓她知道這是一個遊戲，讓她有期待玩遊戲的感覺就好了，我則在旁充當技術指導員，以及故障排除XD，畢竟沒有人能保證寫出來的東西沒有bug 。

照片猜謎


找出一張我們共同去過的地方的照片，利用透明圓圈製造出限定可視範圍的感覺。

選擇餐廳


選擇餐廳，就到這一整棟挑覺得不錯的餐廳，都給它訂位下去，不用擔心這樣很賤，因為當天一定是爆滿的狀態，訂了沒去它也不會理。這時候加入一些搞笑的選項，例如漢堡王、路邊攤小吃。。

用餐時間拼圖遊戲


用餐的時間比較充裕，也可以坐著慢慢完，準備一張我們的合照，開始玩拼圖遊戲。

猜猜用餐金額


這裡比較刺激的是，你不會事先知道用餐金額，因此在你得知用餐金額時，必須要找時間修改金額，好用的Dropbox就派上用場了。

心理測驗（鋪梗）


這裡的重點在要讓女王知道這個測驗只能選擇一次，每一個選項有一個代表的物品，這個物品是接下來遊戲的線索，不同的選項會有不同的線索，但請依照自己的直覺去選擇，如果我跟你都有相同的答案，代表我們心有靈犀一點通～耶！

尋寶遊戲


接下來就是要送給女王的東西啦，由於成功的營造出遊戲氣氛（也是因為它很愛玩），雖然給了幾個提示，但它一個也不想用掉就想到是哪一家了，不過果然還是有點難度，最後在技術指導員的帶領之下終於找到最後的禮物。

後記


稍微運用一點巧思，就可以讓生活增添一些趣味，當然這也是要花時間做的啦，如果覺得這個Idea不錯，想要如法泡製，但又不太會寫的人，可以寄信跟我聯絡，我來幫你客製化，ㄎㄎ～雖然是做給女王的東西，我還是有從中得到一些經驗的。以下就來談談幾個經驗分享。


  1. 資源快取或預存：其實原本我只想做成手機網頁，但實際測試之後發現3G網路連線不夠快，做為互動式遊戲網頁明顯會delay很久，因此我將才將所有的程式包進APP裡頭，執行的速度差蠻多的，雖然作成網頁版有線上即時更新的好處，但是除非你開發的APP應用環境在WIFI裡頭，否則盡可能的將resource做快取或預存。

  2. 詳讀jquery Mobile API再GO：真的可以節省很多開發時間，不過就目前為止，拿它開發靜態網頁非常順，但若要在裡頭嵌入一些額外的js，他的ajax content執行就會有問題。例如在換頁效果中，他是用ajax取得下一頁的內容，而下一頁裡頭有document.ready的執行事件就不會被執行，因此遇到這種情況我必須要加入rel=”external”的設定，讓連結不透過ajax來loading content。這可能要再看一下doc有沒有什麼更好的方式來執行。

  3. 創意創意創意：身為網頁工程師，技術只是工具，創意才是最重要的，上述的技術不難，但加上一些創意與巧思，就可以將冰冷的科技與生活中做互動，進一步讓女王開心，真是一舉兩得。
